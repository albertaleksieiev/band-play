# band play

band-play is a **Google Chrome extension** that provide autoplay on [bandcamp](bandcamp.com)

Check [Chrome Web Store](https://chrome.google.com/webstore/detail/band-play/nooegmjcddclidfdlibmgcpaahkikmlh)
